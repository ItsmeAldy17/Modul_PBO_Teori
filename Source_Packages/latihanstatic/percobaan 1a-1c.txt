1a. Terjadi error pada Main di bagian nama pada System.out.println(Mahasiswa.nama);, karena variabel nama pada class Mahasiswa bersifat non-static, perintah pemanggilan Mahasiswa.nama hanya bisa dijalankan apabila variabel nama bersifat static, seperti pada contoh Mahasiswa.prodi yang tidak terjadi error dikarenakan variabel prodi bersifat static, agar tidak terjadi error pemanggilan nama bisa diawali dengan pemanggilan object terlebih dahulu misalnya object.nama.

1b. Variabel prodi pada object st1 akan ikut berubah seperti variabel prodi st2, karena prodi bersifat static sehingga saat variabel prodi diubah, semua object yang memanggil variabel tersebut akan diganti menjadi Teknologi Rekayasa Perangkat Lunak atau seperti perubahan terakhir yang dilakukan pada saat pemanggilan object st2. Variabel nama tidak berubah seperti variabel prodi, karena variabel nama bersifat non-static. Pada pemanggilan variabel nama juga tidak terjadi error, karena pemanggilan variabel disertai dengan pemanggilan object terlebih dahulu.

1c. Variabel prodi pada object class Mahasiswa semuanya berubah menjadi Software Engineering, karena variabel prodi pada class Mahasiswa bersifat static, sehingga saat pemanggilan Mahasiswa.prodi = "Software Engineering"; seluruh object yang dibuat dari class Mahasiswa yang memanggil variabel prodi, secara kesuluruhan akan diganti menjadi Software Engineering, sedangkan pada variabel nama tidak terjadi perubahan karena variabel nama bersifat non-static.
